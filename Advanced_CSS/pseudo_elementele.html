Ce sunt pseudo-elementele

Aşa cum era cazul cu pseudo-clasele, şi pseudo-elementele sunt continuări speciale, respectiv cuvinte-cheie care se adaugă la sfârşitul selectorilor standard. Prima diferenţă dintre pseudo-clase şi pseudo-elemente constă în modul de formare a acestora. Pentru crearea pseudo-elementelor se foloseşte de două ori caracterul două puncte (::), ca în imaginea de mai jos

Pseudo-elementele au câteva moduri de aplicare diferită. Acestea se pot folosi la:

  • stilizarea unei părţi din textul unui element HTML (prima literă şi prima linie din text);
  • stilizarea anumitor elemente care, iniţial, nu sunt parte a structurii HTML; aşadar, prin utilizarea pseudo-elementelor este permisă adăugarea conţinutului la începutul şi sfârşitul unui element HTML;
  • stilizarea textului selectat.

Ce pseudo-elemente există în CSS?

În limbajul CSS există următoarele pseudo-elemente:

  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::selection

::after

In CSS, pseudo-elementul ::after selectează ultimul descendent virtual asupra elementelor selectate. Cu alte cuvinte, pseudo-elementul ::after adaugă un anumit conţinut la sfârşitul elementului selectat. Sintaxa acestui pseudo-element este următoarea:

        ::after { style properties }

Următorul exemplu va ilustra utilizarea pseudo-elementului ::after. Codul HTML este următorul:

        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. In eu sem vel diam finibus lobortis. 
 In rhoncus neque odio, non maximus ex pretium quis. Vivamus interdum convallis ex ac efficitur.</p>

Pentru ca un conţinut să se adauge la sfârşitul paragrafului prezentat, putem scrie următoarele:

            p::after {
            content: " - Remember this";
            color: red;
            }

astfel textul va arta ca in imaginea de mai jos

::before

Pseudo-elementul ::before adaugă un anumit conţinut la începutul elementului selectat. Cu alte cuvinte, pseudo-elementul ::before are un efect identic pseudo-elementului ::after, doar că există o diferenţă care constă în poziţia pe care se adaugă conţinutul.

Următorul exemplu va ilustra utilizarea pseudo-elementului ::before. Codul HTML este următorul:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. In eu sem vel diam finibus lobortis. In rhoncus neque odio, non maximus ex pretium quis. Vivamus interdum convallis ex ac efficitur.</p>

Pentru adăugarea conţinutului înainte de paragraf, este suficient să scriem in styilizarea CSS:

        p::before {
            content: "Remember this - ";
            color: red;
            }

Textul va arata ca in imaginea de mai jos

::first-letter (:first-letter)

Pseudo-elementul ::first-letter se poate folosi pentru stilizarea primei litere a elementelor selectate. Sintaxa acestui selector este următoarea:

        ::first-letter { style properties }

Următorul exemplu va ilustra utilizarea selectorului ::first-letter. Codul HTML este următorul:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. In eu sem vel diam finibus lobortis. In rhoncus neque odio, non maximus ex pretium quis. Vivamus interdum convallis ex ac efficitur.</p>

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. In eu sem vel diam finibus lobortis. In rhoncus neque odio, non maximus ex pretium quis. Vivamus interdum convallis ex ac efficitur.</p>

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. In eu sem vel diam finibus lobortis. In rhoncus neque odio, non maximus ex pretium quis. Vivamus interdum convallis ex ac efficitur.</p>

Pentru a evidenția prima literă a fiecărui paragraf prin mărirea dimensiunii fontului și schimbarea culorii, putem scrie următoarele:

        p::first-letter { 
            color: red;
            font-size: 130%;
            }

::first-line

Pentru stilizarea primei linii de text a unui element, se poate folosi selectorul CSS cu numele ::first-line. Este vorba de un pseudo-element care asigură doar selectarea primei linii de text a unui element. Textul format din mai multe linii este caracteristic doar pentru elementele block, respectiv pentru elemente ale căror valoare a proprietăţii display este: block, inline-block, table-cell sau table-caption. Pe elementele inline, pseudo-elementul ::first-line nu are efect.

        ::first-line { style properties }

Pseudo-elementul ::first-line selectează textul, aşadar, acesta se poate folosi doar în combinaţie cu proprietăţile referitoare la stilizarea textului. Următorul exemplu ilustrează utilizarea selectorului ::first-line. Codul HTML este următorul:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. In eu sem vel diam finibus lobortis. In rhoncus neque odio, non maximus ex pretium quis. Vivamus interdum convallis ex ac efficitur.</p>

Codul HTML conţine un element de tip paragraph. Cât text se va afla într-o linie pe pagină, depinde de mulţi factori. Totuşi, în felul următor va fi stilizată prima linie de text a paragrafului afişat, indiferent de cantitatea de text care se poate găsi în cadrul ei:

        p::first-line { 
            text-transform: uppercase;
            font-weight: bold;
            }

Aceasta stilizare va arta ca in imaginea de mai jos

::selection

Pseudo-elementele asigură stilizarea unei părţi din textul selectat. Acest lucru se poate obţine datorită pseudo-elementului ::selection. Pentru stilizarea textului selectat, se pot folosi următoarele proprietăţi:

  • color
  • backround-color
  • cursor
  • outline
  • text-decoration
  • text-emphasis-color
  • text-shadow

Sintaxa pseudo-elementului ::selection este următoarea:

        ::selection { style properties }

Următorul exemplu ilustrează utilizarea pseudo-elementului ::selection. Codul HTML este următorul:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. In eu sem vel diam finibus lobortis. In rhoncus neque odio, non maximus ex pretium quis. Vivamus interdum convallis ex ac efficitur.</p>

Stilizarea textului selectat se obţine în felul următor:

        ::selection {
            color: white;
            background: coral;
            }
