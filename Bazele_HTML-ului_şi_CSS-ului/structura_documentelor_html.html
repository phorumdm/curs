Realitatea este totuşi că limbajul HTML, pe lângă elementele care deţin reprezentarea vizuală, recunoaşte şi câteva elemente invizibile, prin care se defineşte structura de bază a paginii HTML.

Familiarizarea cu astfel de elemente s-a amânat, cu un motiv bun, până la această lecţie pentru a înţelege cât mai bine bazele abordate până acum. Totuşi, ca şi un alt oarecare limbaj, şi HTML deţine regulile sale semantice şi sintactice, care trebuie respectate. Una dintre aceste reguli impune că fiecare document HTML trebuie să deţină structura de bază alcătuită din câteva elemente.

Structura documentelor HTML

În structura prezentată putem recunoaşte câteva taguri de deschidere şi de închidere, adică, cu alte cuvinte, câteva elemente HTML. Se poate spune că principala structură a documentului HTML este alcătuită din următoarele părţi:

  • declaraţia DOCTYPE,
  • elementul html,
  • elementul head,
  • elementul body.

DOCTYPE

Primul element al structurii de bază HTML este declaraţia DOCTYPE. Declaraţia DOCTYPE (Document Type Declaration, prescurtat DTD) se află la începutul fiecărei pagini HTML cu „tagul" DOCTYPE. Cuvântul tag este intenţionat pus în ghilimele. De fapt, DOCTYPE nu este un tag HTML real, ci o instrucţiune dată browserului (datorită semnului ! după deschiderea tagului).

Declaraţia DOCTYPE îi comunică browserului versiunea HTML-ului folosită în document. Din lecţia anterioară ne este cunoscut faptul că versiunea curentă a limbajului HTML este 5, ea definind declaraţia în forma următoare:

Elementul HTML

După declaraţia DOCTYPE, structura de bază a documentului HTML defineşte elementul HTML. Având în vedere că DOCTYPE nu se poate denumi drept un element adevărat, putem spune că elementul rădăcină al fiecărui document HTML este un element html. El începe cu tagul "html" de deschidere şi întotdeauna se termină cu tagul de închidere "html":

Câteva dintre cele mai importante caracteristici ale elementului html sunt următoarele:

  • reprezintă elementul rădăcină al documentului HTML;
  • este un element obligatoriu, fără el documentul HTML nu este valid;
  • cuprinde toate celelalte elemente de pe pagină.

Elementul html este părinte pentru doi descendenţi direcţi. Acestea sunt elementele:

  • head – antetul documentului;
  • body – corpul documentului.

Structura de bază a documentelor HTML este un exemplu ideal al formei arborelui. În cadrul elementului html se găsesc elementele head şi body. Este vorba de structura de bază HTML, în timp ce în practică numărul de elemente în acest arbore este mult mai mare.

Corpul documentului HTML – elementul body

Elementul body reprezintă corpul documentului HTML. În cadrul părţii body sunt localizate toate elementele care vrem să fie vizibile pe pagină după parsarea ei în cadrul browserului. Acest lucru înseamnă practic că toate aceste elemente HTML pe care le-am folosit până acum pentru realizarea exemplelor introductive, aparţin corpului documentului şi, de regulă, ele trebuie introduse în cadrul elementului body.

Antetul documentului HTML - elementul head

În cadrul elementului head, care se mai numeşte şi antetul documentului HTML, se introduc elemente care nu sunt direct vizibile pe pagină. De exemplu, în partea head se introduc meta taguri, legături (linkuri) către fişierele CSS şi către alte fişiere etc.

Elementul head începe cu tagul head şi se termină cu tagul /head. Acesta este elementul rădăcină pentru toate datele antetului unuia dintre documentele HTML. Elementul head este elementul părinte cu următoarele elemente HTML:

Elementul title

Elementul title este elementul obligatoriu al fiecărui document HTML. El defineşte denumirea documentului HTML care se foloseşte în câteva situaţii diferite în timpul afişării numelui paginii în:

  • fereastra browserului;
  • paginile favorite;
  • rezultatele căutării din motoarele de căutare.

Un document poate avea un singur element title.

Elementul title se defineşte în cadrul secţiunii head, iar dacă secţiunea head nu este specificată, elementul title se defineşte ca descendent direct al elementului html.

Elementul style

Elementul style se foloseşte pentru definirea stilizării CSS pentru documentul HTML curent. Stilizarea determină cum va arăta elementul HTML. CSS este noţiunea căreia îi vom dedica a doua parte a acestui curs. Într-un document HTML este posibil să avem un număr mai mare de elemente style.

Elementul base

Elementul base defineşte adresa de bază pentru toate adresele relative în cadrul documentului HTML. Rolul acestui element depăşeşte cunoştinţele pe care le deţinem în acest moment, aşadar, la acesta revenim într-una din lecţiile anterioare în care vom vorbi despre linkuri.

Elementul link

Elementul link se foloseşte pentru definirea legăturii cu documentul extern sau cu resursa. Cel mai des se foloseşte pentru legarea fişierului extern cu stilizarea. Elementul link se poate găsi doar în cadrul secţiunii head, de un număr arbitrar de ori. Pentru acest element sunt valabile aceleaşi reguli ca şi pentru elementul anterior. Cu el ne vom ocupa separat în lecţiile despre stilizare.

Elementul meta

Elementul meta se foloseşte pentru definirea metadatelor despre documentul HTML. Metadatele nu se afişează pe pagină, însă de aceea sunt folosite din partea browserului, motorului de căutare (search engine) şi a altor servicii web.

Elementul meta se foloseşte cel mai des pentru definirea:

  • regulilor paginii;
  • cuvintelor-cheie;
  • autorilor documentului;
  • timpului când a fost modificată pagina pentru ultima dată;
  • datelor privind compatibilitatea.

Elementul Meta se scrie întotdeauna în cadrul secţiunii head, de un număr arbitrar de ori. În varianta limbajului HTML5, elementul meta se formează prin utilizarea tagului cu autoînchidere.

Meta elementul poate conţine câteva atribute prin care se editează, suplimentar, scopul său. Cu atributul charset se setează tipul de codare (encoding), ceea ce determină modul în care browserul va citi pagina şi va afişa caracterele specifice. Se recomandă întotdeauna folosirea codării utf-8, în special pentru documentele HTML care conţin părţi din text cu diacritice româneşti, dar şi pentru toate limbile care conţin anumite caractere specifice (germană, spaniolă, norvegiană, daneză etc.). Folosind codarea utf-8, se asigură o afişare adecvată a caracterelor specifice cum ar fi ă,î, ş, ţ â şi altele.

Elementul script

Elementul script se foloseşte pentru integrarea codului JavaScript în documentul HTML. Elementul script poate conţine un cod în mod direct sau poate indica la fişierul extern cu scriptul client.

Elementul nonscript

Elementul noscript se foloseşte pentru definirea unui conţinut alternativ care va fi afişat utilizatorilor în ale căror browsere este dezactivată sau nu este susţinută execuţia scripturilor. Acest element se poate defini în cadrul secţiunii head sau body.
