Cererea HTTP

Home

Pentru a face o anumită cerere serverului respectând regulile protocolului HTTP, definirea adresei URL unde se află resursa solicitată nu este suficientă. Pe lângă URL, este necesară definirea unor informații suplimentare, ambalate într-o formă corespunzătoare. Astfel, cel mai simplu exemplu de cerere HTTP pentru o pagină web poate arăta aşa:

GET /index.html HTTP/1.1

Cererea HTTP prezentată conține toate caracteristicile necesare de care are nevoie serverul pentru a putea forma un răspuns. Dacă cererea prezentată este defalcată în părți, se poate spune că:

GET este numele metodei HTTP;

/index.html este resursa care se solicită;

HTTP/1.1 este versiunea protocolului;

host: www.mysite.com este numele host-ului.

Structura cererii HTTP

Cererea HTTP pe care clientul o trimite serverului este formată din două părți de bază și câteva părți mai mici:

capul(head)

linia de pornire sau linia cererii (start line, request line);

antet (headers);

corp (body).
