Componentele browserului wweb

Home

Componentele de bază pe care le are fiecare browser web sunt:

  • User Interface – mediu de utilizator care are toate elementele vizuale cu care utilizatorul controlează browserul; în acest segment sunt incluse câmpul pentru introducerea adresei, butoanele Back şi Forward și elemente vizuale similare; aranjarea și disponibilitatea diferitelor elemente ale mediului de utilizator variază de la browser la browser;
  • Browser Engine– segmentul care reprezintă legătura dintre mediul de utilizator și componentele pentru randarea conținutului; comenzile utilizatorilor direcționate utilizând elementele din mediul de utilizator prin intermediul acestui segment sunt transmise componentelor pentru randare;
  • Rendering Engine – componenta pentru randare, responsabilă pentru afișarea paginii web; este vorba de o componentă a cărei funcţie este de a crea aspectul unei pagini web pe baza codului HTML și CSS; face acest lucru prin parsarea codului HTML și CSS;
  • Networking – partea browserului responsabilă pentru transmiterea cererilor HTTP și primirea răspunsurilor HTTP;
  • JavaScript Engine – componenta responsabilă cu executarea codului JavaScript; mai devreme, această operaţie a fost realizată prin procesul de interpretare, în timp ce astăzi majoritatea browserelor moderne folosesc o variantă specială de compilare - compilarea JIT (despre ea vom discuta mai multe în cursurile de bază ale JavaScript-ului);
  • UI Backend – componentă responsabilă cu desenarea a tot ce există într-un browser, atât interfața cu utilizatorul, cât și paginile web; aceasta este, de asemenea, o componentă care desenează anumite elemente HTML, cum ar fi checkbox sau elementele radio button sau a listei drop-down;
  • Data Persistence – o componentă responsabilă cu stocarea diferitelor tipuri de date pe un calculator client; browserele moderne, în timpul utilizării, pot produce diferite tipuri de date; acestea pot fi stocate folosind o varietate de mecanisme, cum ar fi: cookie-uri, cache, Web Storage, IndexedDB; astfel de mecanisme de stocare a datelor sunt puse la dispoziția aplicațiilor web care se rulează în interiorul browserului, prin urmare, acestea sunt subiect de interes pentru dezvoltarea front-end; în continuarea acestui program, astfel de mecanisme diferite de stocare a datelor locale vor fi obiectul unui curs separat.
