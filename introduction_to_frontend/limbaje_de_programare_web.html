Limbaje de programare web

Home

În procesul de dezvoltare a software-ului pentru web, astăzi sunt folosite un număr mare de limbaje de programare. Câteva dintre cele mai populare sunt ilustrate în tabelul 4.1.

  1. C#- C# este unul din limbajele create de compania Microsoft; este vorba de un limbaj cu scop general, care este folosit astăzi pe scară largă pentru crearea aplicațiilor desktop, web și mobile; a fost introdus în anul 2000; când este vorba de dezvoltare web, acest limbaj este utilizat pentru partea de server a site-urilor web
  2. PHP- PHP este cu siguranță unul dintre cele mai populare limbaje de programare pentru dezvoltarea web astăzi; îl caracterizează o sintaxă simplă, astfel încât aparține limbajelor care nu sunt greu de învățat; a fost creat în anul 1995, iar creatorul său este Rasmus Lerdorf; este vorba de un limbaj folosit pentru programarea pe server a site-urilor web
  3. Java- Java este considerat, conform multor parametri, drept cel mai popular limbaj de programare astăzi; este destinat programării cu scop general și astfel poate fi folosit pentru a scrie aplicații pentru o gamă foarte largă de dispozitive; acesta este, în acelaşi timp şi cel mai mare avantaj al acestui limbaj, care a fost creat cu scopul ca executarea lui să se facă pe cât mai multe dispozitive computerizate diferite; astăzi Java în mare parte este utilizat pentru programarea web și dezvoltarea aplicațiilor pentru dispozitivele care rulează cu sistemul de operare Android; Java a fost creat în anul 1995, iar creatorul său este James Gosling și compania Sun Microsystems; când este vorba de programare web, Java este utilizat predominant pentru programarea serverului
  4. JavaScript- JavaScript este limbajul de bază al programării client a aplicațiilor web și unul dintre blocurile de construcţie de bază ale web-ului modern; este utilizat pentru a face paginile web mai interactive și este, de asemenea, tehnologia de bază folosită pentru a crea jocuri și animații web și pentru a gestiona datele pe partea clientului a site-urilor web; este partea integrantă a tuturor browserelor web moderne; a apărut pentru prima dată în anul 1995 ca parte integrantă a browserului web Netscape; creatorul lui este Brendan Eich; este vorba de cel mai important limbaj de programare pentru dezvoltarea front-end-urilor
  5. SQL - limbaj al cărui scop principal este gestionarea datelor stocate în baze de date relaționale; astăzi există un număr mare de implementări ale acestui limbaj, create de companii IT de renume; acesta este un limbaj care permite în primul rând citirea, scrierea, ștergerea și modificarea datelor, precum și crearea bazelor de date și tabele, precum și definirea logicii simple, similare limbajelor de programare cu scop general; limbajul a fost creat în anul 1974 de către IBM; cea mai semnificativă variantă a acestui limbaj este cu siguranță MySQL, limbaj ce este utilizat pe scară largă pentru dezvoltarea web; și anume, vizitând un site pe web, există șanse mari ca datele afișate să provină tocmai dintr-o bază de date MySQL
  6. Python – Python este un limbaj de programare cu scop general ale cărui proprietăţi de bază sunt simplitatea sintaxei, lizibilitatea și concizia; acest lucru înseamnă practic că folosind acest limbaj o anumită problemă poate fi rezolvată cu linii de cod semnificativ mai puține decât cu alte limbaje; acest limbaj este folosit astăzi în principal pentru programarea web, ca o alternativă a limbajelor PHP, Java sau C#; a apărut pentru prima dată în anul 1991, iar creatorul său este Guido van Rossum
  7. Perl – un alt limbaj cu scop general care poate fi folosit pentru a scrie diferite tipuri de software; este cel mai utilizat pe web și este cunoscut mai ales pentru bogatele sale capacități de procesare a textelor; a fost creat în anul 1987 de Larry Wall
