Limbajele dezvoltării front-end

Home
  1. HTML – Elemente de construcție principale utilizate de fiecare aplicație sunt datele. Cu alte cuvinte, în cadrul fiecărei aplicații există anumite informații care, într-un mod transparent și organizat, trebuie servite utilizatorului. Situația este similară şi cu site-urile web.Pentru a structura datele site-urilor web este utilizat limbajul HTML.Textul afișat este structurat prin adăugarea anumitor coduri la începutul și sfârșitul lui.Limbajul HTML funcționează folosind coduri de marcare pentru structurarea datelor. Rolul său este astfel dezvăluit şi de numele propriu-zis, care este ascuns în abrevierea HTML - HyperText Markup Language. Cu alte cuvinte, este un limbaj de marcare a textului.Folosind limbajul HTML, este posibil să se definească simplu structura unui document și să se dea semnificații specifice diferitelor date într-un astfel de document. Documentul în care datele sunt marcate cu coduri HTML se numește document HTML. Baza limbajului HTML o reprezintă diverse coduri cu care se marchează datele. Când se spune că cineva cunoaște limbajul HTML, de fapt înseamnă că acela cunoaște diferiți markeri cu care marchează datele într-un document HTML.Browserele web determină sensul semantic al conținutului pe baza codurilor din cadrul documentelor HTML. Astfel, ei știu că un anumit bloc de text este, de fapt, titlul și un alt bloc de text este paragraf. Cele mai populare și mai utilizate browsere web sunt Chrome, Safari, Firefox, Opera, Internet Explorer.
  2. CSS– CSS este un alt limbaj web de bază și dezvoltare front-end. CSS definește prezentarea, respectiv aspectul documentelor HTML.CSS descrie, respectiv modifică aspectul și formatarea tuturor elementelor dintr-un document HTML.

JavaScript - Pe lângă cele două limbaje de calculator anterioare, dezvoltarea front-end este caracterizată şi prin utilizarea limbajului de programare JavaScript.JavaScript este un limbaj de programare pentru crearea logicii de prezentare a site-urilor web. Ce înseamnă practic acest lucru? JavaScript este un limbaj de programare care înseamnă, practic, că permite definirea logicii de programare. Deoarece se execută direct în browser, acest lucru permite ca orice calculator client individual care accesează un site web să se transforme într-o scenă pentru executarea codului. Un asemenea lucru deschide ușa către manipulări puternice ale conținutului site-ului web. Astfel, utilizarea limbajului JavaScript poate afecta datele site-ului web, restricția fiind imaginația programatorului care creează codul de programare al limbajului JavaScript.Una dintre cele mai importante funcţionalităţi ale limbajului JavaScript este posibilitatea de a comunica direct cu serverul web în fundal, fără a fi nevoie de reîmprospătarea paginii site-ului web. În felul acesta, JavaScript permite ca site-urile web să fie mai dinamice și mai interactive și conform proprietăţilor să fie mai apropiate de programele reale de calculator.
  4. Browserul web – Browserul web este un program (aplicație) conceput pentru a accesa informații pe World Wide Web.Browserul web este direct responsabil de gestionarea codului scris în limbajele HTML, CSS și JS.Şi anume, în browser se face parsarea codului limbajelor HTML și CSS. Simplu spus, acest lucru înseamnă că browserul web citește mai întâi conținutul HTML al unei pagini web. Pe baza diferitelor coduri ce există în limbajul HTML, browserul formează structura de bază a paginii web.Pe lângă parsarea codului limbajelor HTML și CSS, browserul web este responsabil și de executarea codului JavaScript. Când am vorbit despre limbajele HTML și CSS, intenţionat a fost omis acest termen de executare, deoarece, așa cum s-a spus deja, HTML și CSS nu sunt limbaje de programare și nu se poate spune că se execută. JavaScript, pe de altă parte, este un limbaj de programare care necesită existenţa unui browser web. Browserul web este responsabil pentru convertirea codului JavaScript în codul mașinii, deoarece este singurul limbaj pe care procesorul calculatorului îl înțelege. O astfel de conversie se numește altfel compilare și se face prin intermediul componentelor speciale ale browserului web.
