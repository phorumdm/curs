Pentru a face mai interesant acest prim exemplu de creare a documentului HTML, în rândurile următoare vom face stilizarea lui de bază. În lecțiile anterioare s-a spus că pentru stilizarea unui document HTML este folosit limbajul CSS (Cascading Style Sheets). Cu sintaxa și caracteristicile acestui limbaj ne vom familiariza în cursul următor, iar acum, fără a intra în detalii în regulile CSS, vom face o stilizare de bază a primului nostru document HTML..

Textul celor trei elemente care există în cadrul listei le vom colora cu trei culori diferite - roșu, verde și albastru. Vom realiza acest lucru în felul următor:

<h1>Introduction to Front-end Development</h1> <p>A front-end developer architects and develops client part of websites and applications using web technologies (languages):</p> <ul> <li style="color: red;">HTML</li> <li style="color: green;">CSS</li> <li style="color: blue;">JavaScript</li> </ul>
